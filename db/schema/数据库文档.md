## 数据库结构

### 表结构

#### 1. rules（规则主表）
- **记录数**: 4,939条
- **字段**:
  - `id`: 主键，自增
  - `rule_name`: 规则名称
  - `rule_type`: 规则类型（yara/sigma）
  - `category`: 规则分类
  - `file_path`: 规则文件路径
  - `file_hash`: 文件哈希（SHA256）
  - `status`: 状态（active/disabled/deprecated）
  - `priority`: 优先级（1-10）
  - `created_at`: 创建时间
  - `updated_at`: 更新时间
  - `version`: 版本号

#### 2. rule_metadata（规则元数据表）
- **记录数**: 4,939条
- **覆盖率**: 100%
- **字段**:
  - `rule_id`: 外键，关联rules表
  - `description`: 规则描述
  - `author`: 作者
  - `source`: 来源
  - `mitre_attack_tags`: MITRE ATT&CK标签（JSON格式）
  - `platform`: 平台
  - `threat_type`: 威胁类型
  - `false_positive_rate`: 误报率
  - `detection_rate`: 检测率

#### 3. rule_index（规则索引表）
- **记录数**: 15,930条
- **用途**: 提供快速搜索功能
- **字段**:
  - `id`: 主键
  - `rule_id`: 外键，关联rules表
  - `keyword`: 关键词
  - `index_type`: 索引类型（name/tag/category/platform/threat）

#### 4. rule_versions（规则版本历史表）
- **记录数**: 0条（暂未使用）
- **用途**: 跟踪规则版本变更历史

#### 5. rule_statistics（规则使用统计表）
- **记录数**: 0条（暂未使用）
- **用途**: 记录规则使用频率和匹配统计

### 索引
- `idx_rule_type`: 规则类型索引
- `idx_category`: 分类索引
- `idx_status`: 状态索引
- `idx_priority`: 优先级索引
- `idx_rule_name`: 规则名称索引
- `idx_keyword`: 关键词索引
- `idx_index_type`: 索引类型索引

## 使用指南

### 查询数据库

#### 使用Python脚本查询
```bash
# 查看数据库统计信息
python query_database.py

# 检查数据库状态
python check_db_status.py
```

#### 使用SQL直接查询
```bash
# 打开SQLite命令行
sqlite3 rules_database.db

# 示例查询
SELECT COUNT(*) FROM rules WHERE rule_type = 'yara';
SELECT * FROM rules WHERE category = 'malware' LIMIT 10;
SELECT r.rule_name, rm.description FROM rules r 
JOIN rule_metadata rm ON r.id = rm.rule_id 
WHERE r.rule_type = 'yara' LIMIT 10;
```

### 导入新规则

```bash
# 导入所有规则
python import_rules.py --all

# 只导入YARA规则
python import_rules.py --yara yara_rules/organized

# 只导入Sigma规则
python import_rules.py --sigma sigma_all_rules/rules
```

### 初始化数据库

```bash
# 初始化数据库（如果数据库不存在）
python init_rules_database.py
```
